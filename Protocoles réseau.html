Protocoles réseau

1. TCP/IP

Signifie : Transmission Control Protocol / Internet Protocol
Base de fonctionnement d’Internet
Permet l’adressage IP et l’envoi des données

2. HTTP

Signifie : HyperText Transfer Protocol
Utilisé pour consulter des pages web
Fonctionne avec les navigateurs (Chrome, Firefox…).

3.HTTPS

Version sécurisée de HTTP
Utilise le chiffrement (SSL/TLS)
